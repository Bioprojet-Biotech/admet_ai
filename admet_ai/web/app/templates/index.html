ADMET-AI

ADMET-AI is a simple, fast, and accurate web interface for predicting the Absorption, Distribution, Metabolism, Excretion, and Toxicity (ADMET) properties of molecules using machine learning models.

ADMET: The evaluation of pharmacokinetics and toxicity is crucial for the design of new therapeutic candidates. In silico virtual screens and generative AI output a vast number of molecules that must be filtered to a tractable number for synthesis and experimental validation. An effective primary filter is to evaluate candidate compounds based on their Absorption, Distribution, Metabolism, Excretion, and Toxicity (ADMET) properties.

Models: ADMET-AI uses a graph neural network called Chemprop-RDKit, which was trained on 41 ADMET datasets from the Therapeutics Data Commons (TDC). ADMET-AI’s Chemprop-RDKit model has the highest average rank on the TDC ADMET Benchmark Group leaderboard and, it is also currently the fastest web-based ADMET predictor.

References: Details regarding the training and evaluation of the model will be described in a forthcoming paper.


Input

Molecules: ADMET-AI can make predictions for up to {{ "{:,}".format(max_molecules) }} molecules at a time by (1) providing SMILES (one per line) in the text box, (2) uploading a CSV file with SMILES, or (3) drawing a molecule using the interactive tool and then converting it to a SMILES.

DrugBank Reference: To provide relevant context, ADMET predictions on input molecules are compared to predictions on approved drugs from the DrugBank. This reference set can be filtered to a specific category of drugs for a more relevant comparison by selecting an Anatomical Therapeutic Chemical (ATC) code.

Predict: After selecting input molecules and a DrugBank reference set, click the “Predict” button to make ADMET predictions.

Output

Summary Plot: The summary plot shows the distribution of ADMET predictions for all input molecules compared to the DrugBank reference set. The x- and y-axes can be changed to show any two ADMET properties.

Radial Plot: For each input molecule, a radial plot is shown summarizing five key ADMET properties in terms of their DrugBank percentile:

  1. Blood-Brain Barrier Safe: The probability that the molecule does not cross the blood-brain barrier (i.e., 1 - Blood-Brain Barrier Penetration percentile).
  2. hERG Safe: The probability that the molecule does not block the human ether-a-go-go (hERG) channel (i.e., 1 - hERG Blocking percentile).
  3. Bioavailable: The probability that the molecule is orally bioavailable (i.e., Oral Bioavailability percentile).
  4. Soluble: The aqueous solubility of the molecule (i.e., Aqueous Solubility percentile).
  5. Non-Toxic: The probability that the molecule is not toxic across all toxicity properties (i.e., 1 - maximum Toxicity percentile).

ADMET Predictions: Clicking on each displayed molecule will show the molecule’s ADMET predictions in tabular form. For each molecule, ADMET-AI computes 8 physicochemical properties using RDKit and predicts 41 ADMET properties using its Chemprop-RDKit graph neural network. For regression properties, the property is directly predicted with the units shown. For classification properties, the predicted value is the probability that the molecule has the property (e.g., probability of blood-brain barrier penetration). Additionally, the percentile of the molecule’s property compared to the DrugBank reference is shown.

Download Predictions: Predictions for the first {{ "{:,}".format(max_visible_molecules) }} molecules are shown on the website. Predictions for all input molecules can be downloaded as a CSV file by clicking the “Download Results” button.

Notes

Storing Molecules: ADMET-AI does not store any molecules uploaded to the website. All molecules are deleted after predictions are made.


ADMET-AI can be run locally as a command line tool for large-scale batch prediction or as a Python module for use within other Python-based drug discovery tools. Please see https://github.com/swansonk14/admet_ai for more details.
